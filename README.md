# crypto-quant

加密货币量化交易项目。

## 项目结构

```
crypto-quant/
├── data/           # 数据获取与存储
├── strategies/     # 交易策略
├── backtest/       # 回测引擎
├── execution/      # 下单执行
├── utils/          # 工具函数
└── tests/          # 测试
```

## 环境

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
